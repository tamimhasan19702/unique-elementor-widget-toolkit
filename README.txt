== Description ==
Unique Sphere Widget Toolkit is a WordPress plugin that adds a variety of widgets to Elementor for enhancing your website's functionality and appearance. The plugin is designed to be lightweight, easy to use, and compatible with the latest version of Elementor.

== Installation ==

1. Upload `unique-sphere-widget-toolkit.php` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings as needed.

== Configuration ==

To use the plugin, follow these steps:

1. Go to **Elementor > Widgets** in WordPress.
2. The plugin will add a variety of unique and innovative widgets to the Elementor widget panel.
3. Use the widgets to enhance your website's functionality and appearance.
